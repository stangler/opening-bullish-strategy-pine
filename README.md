
# 寄付始値エントリー 1本ホールド戦略 [Pine Script v6]

09:00（寄り付き）の**始値で即エントリー**し、**最初の5分足確定（09:05）で即エグジット**する統計検証用ストラテジー。陽線判定を待たずにエントリーし、その足が結果的に陽線になったか、利益が出たかを検証。

## 特徴

- **時間足**: 5分足専用
- **エントリー**: 09:00 寄り付き始値（成行）
- **エグジット**: 09:05 最初の足の確定値（成行）
- **目的**: 寄り付き〜09:05の純粋な「陽線化確率」と「期待値」の測定
- **期間指定**: 2026/04/01 〜 本日（デフォルト設定）
- **精度**: `process_orders_on_close = true` により足確定時の執行を最適化

## 戦略のロジック

1. **予約**: 前日最終足で当日09:00の「成行買い」を予約（`timeframe.change("D")[1]`）
2. **約定**: 当日09:00始値でロング
3. **決済**: 当日09:05終値でクローズ（`timeframe.change("D")`）
4. **可視化**: 決済後、その足が陽線（close > open）なら「陽線」、陰線なら「陰線」とラベル表示

## 使い方

1. TradingView **5分足** チャート表示
2. Pineエディタにコード貼付
3. 「チャートに追加」
4. 戦略テスターで「純利益」「勝率（陽線確率）」確認

### 推奨設定

- **手数料/スリッページ**: 成行のため、設定画面でスリッページ 1〜2 程度を推奨
- **期間**: 2026年4月以降の直近データ

## スクリプトファイル

- `寄付0900入_0905出.pine`

## 注意事項

- **陽線判定後のエントリーではない**: 陽線になる前に買うため、リスクあり
- **検証用**: どの銘柄が「寄付陽線」になりやすいかのアノマリー発見に特化
- **オーバーナイト回避**: 15:30以降の持ち越しはしない仕様
