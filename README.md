# 寄付始値エントリー 1本ホールド戦略 [Pine Script v6]

09:00（寄り付き）の**始値で即エントリー**し、**最初の1分足／3分足／5分足確定（09:01／09:03／09:05）で即エグジット**する統計検証用ストラテジー。  
陽線判定を待たずにエントリーし、その足が結果的に陽線になったか、利益が出たかを検証。

## 特徴

- **時間足**: 1分足・3分足・5分足に対応
- **エントリー**: 09:00 寄り付き始値（成行）
- **エグジット**: 09:01／09:03／09:05 の最初の足の確定値（成行）
- **ギャップ別統計**: ギャップアップ／ギャップダウン／続き足の3パターンに分けて陽線・陰線本数を集計
- **目的**: 寄り付き〜09:01/03/05の純粋な「陽線化確率」と「期待値」の測定（パターン別）
- **期間指定**: 2026/04/01 〜 本日（デフォルト設定）
- **精度**: `process_orders_on_close = true` により足確定時の執行を最適化

## 戦略のロジック

1. **予約**: 前日最終足で当日09:00の「成行買い」を予約（`timeframe.change("D")[1]`）
2. **約定**: 当日09:00始値でロング
3. **決済**: 当日最初の1分足／3分足／5分足の終値でクローズ（`timeframe.change("D")`）
4. **ギャップ判定**: 前日終値と当日始値の差で「ギャップアップ／ダウン／続き足」を判定
5. **可視化**: 決済後、その足が陽線（close > open）なら「陽線」、陰線なら「陰線」とラベル表示
6. **統計**: パターン別に陽線・陰線本数をカウントし、最終足で集計表示

## 使い方

1. TradingViewで **1分足・3分足・5分足** のいずれかにチャートを切り替え
2. Pineエディタにコードを貼り付け
3. 「チャートに追加」
4. 戦略テスターで「純利益」「勝率（陽線確率）」「ギャップ別統計」を確認

### 推奨設定

- **手数料/スリッページ**: 成行のため、設定画面でスリッページ 1〜2 程度を推奨
- **期間**: 2026年4月以降の直近データ
- **時間足選択**: 入力設定で「1分足／3分足／5分足」を現在のチャートに合わせて設定（UI上のヒント）

## スクリプトファイル

- `寄付0900入_090105出_ギャップ別統計.pine`

## 注意事項

- **陽線判定後のエントリーではない**: 陽線になる前に買うため、リスクあり
- **検証用**: どの銘柄が「寄付陽線」になりやすいか、どのギャップパターンが強いかのアノマリー発見に特化
- **オーバーナイト回避**: 当日最初の足で必ず決済するため、オーバーナイト持ち越しはしない仕様
