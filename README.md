# 寄付陽線 1本ホールド戦略 [Pine Script v6]

5分足で**寄付（その日の最初の取引足）が陽線**の場合に成行ロングでエントリーし、**次の足確定時に成行でエグジット**するシンプルな日本株向けストラテジーです。

遅い寄り付き（9:00に取引がない場合）でも、その日最初の足が陽線なら自動的に対応します。

## 特徴

- **時間足**: 5分足専用
- **エントリー**: その日の初足が陽線（close > open）で確定した瞬間に成行ロング
- **エグジット**: 次の5分足が確定した瞬間に成行全決済（実質1本だけホールド）
- **対応**: 9:00寄り付きだけでなく、遅れて寄り付いた場合も自動検出
- **期間指定**: バックテスト期間を入力で簡単に設定可能（デフォルト: 2026/4/1 〜 本日）
- **取引セッション**: デフォルト「0900-1530」（日本株標準）
- Pine Script **Version 6** 対応

## 戦略のロジック

1. その日の最初の5分足を検出（`timeframe.change("1D")` + セッション内）
2. 陽線か判定（`close > open`）
3. 期間内であればエントリー
4. 次の足で必ずエグジット

非常にシンプルで、寄付直後の勢いを1本だけ捉えることを目的としています。

## 使い方

1. TradingViewで **5分足** の日本株銘柄を表示
2. Pine Editorを開き、このリポジトリの `strategy.pine`（またはスクリプトファイル）をコピー＆ペースト
3. 「Add to Chart」をクリック
4. 戦略テスターでバックテストを実行

### 入力パラメータ（おすすめ設定）

- **取引セッション**: `0900-1530`（デフォルト）
- **バックテスト期間**:
  - 開始: 2026年4月1日
  - 終了: 今日の日付に調整（終了日を手動で設定）
- **寄付初陽線のみエントリー**: ON（デフォルト）

## スクリプトファイル

- `寄付陽線1本ホールド.pine` （または `strategy.pine`）

## 注意事項

- このストラテジーは**バックテスト・アイデア検証用**です。リアルトレードではスリッページ、手数料、市場状況を十分考慮してください。
- 寄付直後はボラティリティが高いため、実際の運用時には損切りを追加することを強く推奨します。
- 日本株以外（米国株など）で使う場合はセッション設定を変更してください。
- 過去のパフォーマンスは将来の結果を保証するものではありません。

## カスタマイズしたい場合

- 損切り（ストップロス）や利益確定を追加したい
- 出来高フィルターや前日比フィルターを入れたい
- エグジットタイミングを変更したい
